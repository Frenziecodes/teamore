Create parent-managed WooCommerce subaccounts for teams, families, companies, and repeat buyers.

== Description ==

Customer Subaccounts for WooCommerce helps your store feel easier to buy from when one customer needs more than one person involved.

Some customers are not just one person. They are a purchasing team, a family, an office, a school, a branch, or a company account with people placing orders under the same relationship. This plugin gives those customers a simple way to create and manage subaccounts from WooCommerce My Account, while giving store admins clear visibility into who belongs to whom.

Instead of forcing every buyer into a separate, disconnected customer account, you can let a parent customer manage the people connected to them. Subaccounts remain normal WordPress users with the WooCommerce customer role, so the store stays familiar while the buying experience feels more organized.

Use it to make repeat ordering smoother, reduce account confusion, and give your best customers a cleaner way to manage the people who buy with them.

= Built for real customer relationships =

Customer Subaccounts for WooCommerce is useful for stores that serve:

* Businesses with employees who place orders.
* Schools, clubs, and organizations with multiple buyers.
* Families or households that want connected customer accounts.
* Wholesale or repeat customers who need simple account delegation.
* Stores that want clearer parent and subaccount visibility in WooCommerce.

= What customers can do =

Parent customers can create subaccounts, manage basic details, view connected subaccounts from My Account, see orders placed by subaccounts, and optionally switch into a subaccount session when enabled.

Subaccounts can shop as regular WooCommerce customers while staying connected to the parent account that created them.

= What store admins can do =

Store admins can decide whether subaccounts are created by admins, customers, or both. Admins can set a subaccount limit, choose the account label style, configure display names, and show parent/subaccount context in order and user screens.

== Features ==

* Parent-managed subaccounts inside WooCommerce My Account.
* Subaccounts use normal WordPress customer accounts.
* Parent customers can create, edit, and delete their own subaccounts.
* Admins can create subaccounts from customer profile screens.
* Configurable subaccount limit, including unlimited subaccounts.
* Creation controls for admins only, customers only, or both.
* Multi-User Mode and Sub-User Mode labels.
* Configurable display name format for cleaner admin screens.
* Parent accounts can view orders placed by subaccounts.
* My Account order lists can show who placed each order.
* Optional parent/subaccount information on WooCommerce order pages.
* Optional parent/subaccount information in WooCommerce order lists.
* Optional parent/subaccount information in the WordPress users list.
* Optional parent-to-subaccount user switching.
* WooCommerce High-Performance Order Storage compatibility declaration.

== Installation ==

1. Upload the `customer-subaccounts-for-woocommerce` folder to `/wp-content/plugins/`.
2. Activate Customer Subaccounts for WooCommerce from the Plugins screen.
3. Go to WooCommerce > Customer Subaccounts for WooCommerce.
4. Review the subaccount settings and choose who can create subaccounts.
5. If the My Account endpoint does not appear, visit Settings > Permalinks once to refresh rewrite rules.

== Frequently Asked Questions ==

= Does this plugin create a custom user type? =

No. Subaccounts are normal WordPress users with the WooCommerce customer role. That keeps account behavior familiar and compatible with standard WooCommerce customer flows.

= Can customers create their own subaccounts? =

Yes, if you enable customer creation in the plugin settings. You can also limit creation to admins, or allow both admins and customers to create subaccounts.

= Can I limit how many subaccounts a parent customer can create? =

Yes. You can set a maximum number of subaccounts per parent customer. You can also enter 0 to allow unlimited subaccounts.

= Can subaccounts manage other subaccounts? =

No. Subaccounts cannot manage other accounts. Parent customers remain in control of the subaccounts connected to them.

= Can parent customers see subaccount orders? =

Yes. Parent customers can see orders placed by their subaccounts, helping them understand account activity without asking store staff to investigate.

= Can store admins see parent and subaccount relationships? =

Yes. You can show parent/subaccount information on WooCommerce order pages, WooCommerce order lists, and the WordPress users list.

= Does the plugin support user switching? =

Yes. When enabled, parent customers can switch into a subaccount session from My Account and switch back to the parent account.

= Does it work with WooCommerce High-Performance Order Storage? =

The plugin declares compatibility with WooCommerce custom order tables.
